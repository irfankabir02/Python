## SYSTEM CAPABILITIES

### What It Does

**Input**: Python repository path  
**Process**:
1. Analyzes code for classes, errors, and patterns
2. Maps programming concepts to physical comedy gags
3. Generates 90-second narrative structure
4. Compresses prompt to fit token budget
5. Outputs production-ready Sora 2 prompt

**Output**: Black & white silent-film video prompt (Chaplin-inspired)

## QUICK START

### Installation

Copy the three core modules into your project:
```
your_project/
├── sora_prompt_generator.py
├── sora_client.py
└── sora_integration.py
```

### Basic Usage

```python
from sora_prompt_generator import build_prompt_from_repo, TokenBudgetLevel
from pathlib import Path

# Generate prompt from repository
prompt = build_prompt_from_repo(
    repo_path=Path("your/repo/path"),
    budget=TokenBudgetLevel.LOW  # or ULTRA_LOW, MEDIUM, STANDARD
)

print(prompt)  # Ready for Sora 2 API
```

### CLI Usage

```bash
python sora_integration.py \
  --repo "path/to/repo" \
  --budget LOW \
  --dry-run
```
