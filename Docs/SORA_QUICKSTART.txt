## ðŸ’» Python API Quick Reference

### Generate Prompt
```python
from sora_prompt_generator import build_prompt_from_repo, TokenBudgetLevel

prompt = build_prompt_from_repo(
    repo_path=Path("e:/projects/python/hello"),
    budget=TokenBudgetLevel.LOW,  # or MEDIUM, STANDARD, ULTRA_LOW
)
```

### Create Client
```python
from sora_client import SoraClient

client = SoraClient(
    api_key="sk-...",
    monthly_budget_usd=50.0,
)
```

### Generate Video
```python
from sora_client import VideoGenerationRequest

request = VideoGenerationRequest(
    prompt=prompt,
    duration=90.0,
    quality="high",  # or "low", "medium"
)

# Check budget first
if client.check_budget(request.estimate_cost_usd()):
    response = client.generate_video(request)
    print(f"Video ID: {response.video_id}")
```

### Poll Status
```python
response = client.get_video_status(video_id)
print(f"Status: {response.status}")  # pending, processing, completed, failed
```
