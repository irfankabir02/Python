## QUICK START GUIDE

### Installation

Copy the three core modules into your project:

```bash
cp sora_prompt_generator.py your_project/
cp sora_client.py your_project/
cp sora_integration.py your_project/
```

### Basic Usage

```python
from sora_prompt_generator import build_prompt_from_repo, TokenBudgetLevel
from pathlib import Path

# Generate prompt
prompt = build_prompt_from_repo(
    repo_path=Path("your/repo/path"),
    budget=TokenBudgetLevel.LOW
)

# Use with Sora 2 API
print(prompt)
```

### CLI Usage

```bash
python sora_integration.py \
  --repo "path/to/your/repo" \
  --budget LOW \
  --dry-run
```
