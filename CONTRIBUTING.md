# Contributing

Thank you for your interest in contributing to this repository.

- Run tests: `pytest`
- Run type checks: `mypy --strict src/`
- Run linters/formatters: `flake8`, `black`, `isort`
- Ensure docs remain accurate (especially `README.md`, `Docs/` Sora guides).
- Use conventional commit messages where possible (e.g. `feat:`, `fix:`, `docs:`).

Please open an issue or draft PR if you are unsure about a change that affects the Sora video system or hello-app CLI.
